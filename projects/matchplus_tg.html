Match+ Telegram Bot

  • Position: Developer
  • Technologies Used: Python, MongoDB, BeautifulSoup, ChatGPT
  • Github: Link
  • Telegram: Link

DESCRIPTION:

The goal of this project is to create an assistant for football enthusiasts and those who want to stay updated with scores without needing an additional app! This project used the Python library BeautifulSoup to scrape football match dates, times, and scores from leading websites around the world. This information was saved to a MongoDB database and sent to our "Match+" Telegram group using a "schedule" mechanism through the created Telegram bots.


FEATURES:

  • Instant Notifications: The bot immediately informs you before, during, and after the matches of the most popular leagues in the world.

  • Wide League Pool: Supports the most loved football leagues worldwide and provides you with extensive options.

  • Match Details: Getting information about matches is very easy with the Telegram football bot. You can learn about dates and times, scores, the day's schedule, the day's results, and more.
